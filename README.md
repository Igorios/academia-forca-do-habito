
# Academia Força do Hábito
## Versões do projeto:

* php 7.4.26
* composer 2.2.6
* laravel 8

## Instruções
* clone o repositório:
~~~~
 $ git clone https://github.com/Igorios/academia-forca-do-habito.git
~~~~

* Executar o projeto no terminal
~~~~
php artisan serve
~~~~
### Abrir no browser: http://127.0.0.1:8000/

#

* Rodar migrações
~~~~
php artisan migrate
~~~~
